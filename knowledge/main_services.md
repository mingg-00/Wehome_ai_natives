# 위홈 주요 서비스

## 게스트(여행자) 서비스

### 숙소 검색 및 예약
- 지역, 날짜, 인원수 기반 검색
- 지도 기반 위치 확인
- 실시간 가격 비교
- 즉시 예약 또는 요청 예약 선택 가능

### 큐레이션 테마
- 제주 한달살기 특화 숙소
- 반려동물 동반 가능 숙소
- 바다 뷰 숙소
- 풀빌라·독채 숙소
- 가족 여행 맞춤 숙소 (4인 이상)

### 게스트 보호
- 여행자 보험 기본 제공
- 체크인 24시간 전 취소 시 전액 환불
- 분쟁 발생 시 위홈 중재 서비스

## 호스트(집주인) 서비스

### 수익 관리
- 자동 정산 (예약 완료 후 D+2 입금)
- 월별 수익 리포트 제공
- 세금계산서 발행 지원

### 운영 지원
- 전문 사진 촬영 서비스 (서울·제주 한정)
- 청소업체 연결 서비스
- 스마트 도어락 연동 가이드
- 가격 자동 최적화 도구 (AI 기반)

### 마케팅 지원
- 위홈 공식 SNS 노출
- 계절별 프로모션 참여
- 검색 상위 노출 관리

## 기업·단체 서비스
- 법인 장기 계약 숙소 제공
- 워케이션 패키지 (회사 단체 예약)
- 해외 기업 임직원 한국 체류 숙박 지원
